Para iniciar a aplicação, basta colocar toda a pasta GameWare no Workspace do Eclipse, e fazer o seguinte:

File -> Import -> General -> Existing Projects into Workspace -> Select Root Directory (Seleccionar a pasta do Projecto) -> Finish

Logo de seguida, é fácilo correr e testar o programa numa máquina virtual. No entanto, Android SDK, Eclipse ADT Plugin e também uma máquina virtual com a versão 2.1 ou mais é necessária.

-----//----

Em alternativa, é possível passar o .apk para o telemóvel, instalar, e a aplicação estará pronta a correr.
